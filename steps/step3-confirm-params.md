# Step 3: 确认续写参数

## 执行清单

- [ ] 确认续写章数
- [ ] 确认每章字数
- [ ] 确认情节方向
- [ ] 确认焦点角色
- [ ] 确认风格要求
- [ ] 计算并告知总字数（续写章数 × 每章字数）

## 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| 续写章数 | 总共续写多少章 | 10章 / 20章 / 50章 |
| 每章字数 | 每章大约多少字 | 2000字 / 3000字 / 5000字 |
| 情节方向 | 续写的大致走向 | 自由发展 / 指定方向 / 指定冲突 |
| 焦点角色 | 重点关注的角色 | 延续原文 / 指定角色 |
| 风格要求 | 是否有特殊风格偏好 | 保持原文 / 略微调整 |
