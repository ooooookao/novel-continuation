# Step 4: 生成续写大纲

## 执行清单

- [ ] 根据原文大纲 + 用户参数生成续写大纲
- [ ] 大纲详细程度与字数匹配（≤3000字/章：3-5行；3000-8000：场景级；≥8000：场景拆分）
- [ ] 续写 > 20 章？→ 大纲按弧拆分文件
- [ ] 完成后进入 Step 5

根据原文大纲和用户参数，生成续写大纲。

**续写大纲模板**: [assets/templates/outline-continuation.md](../assets/templates/outline-continuation.md)

## 大纲要求

- 每章大纲的详细程度应与每章字数匹配：
  - ≤3000字/章：每章 3-5 行概要即可
  - 3000-8000字/章：每章需列出场景、关键事件、角色互动
  - ≥8000字/章：每章应拆分为多个场景，标注每个场景的地点、人物、事件、冲突、转折
- 标注出场角色和推进的情节线
- 安排节奏起伏，埋新伏笔同时安排旧伏笔揭晓

## 大纲拆分（续写 > 20 章）

续写超过 20 章时，大纲按弧线拆分为多个文件：

```
outline-continuation/
├── overview.md              # 总览：整体构思、节奏曲线、伏笔计划
├── arc-01-ch01-ch10.md      # 第1-10章大纲
├── arc-02-ch11-ch20.md      # 第11-20章大纲
├── arc-03-ch21-ch30.md      # 第21-30章大纲
└── ...
```

每个弧线文件包含该段所有章节的场景级大纲。这样写作时只需读取当前弧线的大纲，不用加载全部。
