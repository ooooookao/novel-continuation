# Step 9: 一键导出完整小说

**触发**: 用户说"导出小说"、"合并所有章节"、"生成完整文件"

## 执行清单

- [ ] 读取所有章节按序拼接（章节间空两行，每章加标题）
- [ ] 生成统计信息（总章数/总字数/平均/最长/最短/评分趋势/伏笔统计）
- [ ] 保存为 `output/novel-full.txt` 和 `.md` 格式
- [ ] 告知用户导出路径

## 统计信息模板

```
=== 写作统计 ===

- 项目名称: [原文标题] — 续写
- 续写范围: 第X章 - 第Y章
- 总章数: X 章
- 总字数: X 字
- 平均每章: X 字
- 最长章节: 第X章 (X字)
- 最短章节: 第X章 (X字)
- 质量评分趋势: X.X → X.X → X.X
- 伏笔统计: X 个活跃 / X 个已揭晓 / X 个被遗忘
- 项目启动: YYYY-MM-DD
- 导出时间: YYYY-MM-DD
```

## 导出格式

- `.txt` — 纯文本，章节间空两行
- `.md` — Markdown 格式，章节标题用 `#` 标记
- 如需 `.docx`：`pandoc novel-full.md -o novel-full.docx`
